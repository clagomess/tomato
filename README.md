# Tomato

## Data directory structure

```
~/.tomato/
├── data/
│   └── data-session.json
│   └── workspace-{ID}/
│       └── workspace-{ID}.json
│       └── environment-{ID}.json
│       └── workspace-session.json
│       └── collection-{ID}/
│           └── collection-{ID}.json
│           └── request-{ID}.json
│           └── collection-{ID}/
│               └── collection-{ID}.json
│               └── request-{ID}.json
│       └── request-{ID}.json
└── configuration.json
└── cache/
```
